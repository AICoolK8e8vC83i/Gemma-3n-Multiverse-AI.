## ğŸš€ **Getting Started**

### **Prerequisites**
- Python 3.8+
- Node.js 16+
- Google AI Studio API key
- Google Pixel 9a (for full experience)

### **Installation**

```bash
# Clone the repository
git clone https://github.com/AICoolK8e8vC83i/Gemma-3n-Multiverse-AI..git
cd Gemma-3n-Multiverse-AI.

# Set up Python environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

# Set up mobile app
cd Gemma3nMobile
npm install
```

### **Configuration**

1. **Create `.env` file:**
```bash
GEMINI_API_KEY=your_google_ai_studio_api_key_here
```

2. **Start the sync server:**
```bash
python sync_server.py
```

3. **Launch Streamlit:**
```bash
streamlit run streamlit.py --server.port 8501 --server.address 0.0.0.0
```

4. **Start mobile app:**
```bash
cd Gemma3nMobile
npm start
```

5. **Connect Pixel 9a:**
- Scan QR code with Expo Go
- App will connect to local network automatically
